The Golden Hard Drive

This page outlines the methodology behind my work-in-progress personal goal to catalog a comprehensive collection of lossless music.

My current target is 1,500 albums, at which point the library will be sufficiently large given the amount of music I listen to and would like to include.

The genres included in this collection are limited to my personal taste (rock, jazz, soul, funk, hip-hop, electronic, ambient, and world, among others).

Once completed, I will transfer everything to a limited number of gold-painted HDDs.

Folder directory structure:

/Library/Artist/Album/

*Multi-CD releases are further divided into separate folders by disc number.

File naming scheme:

Artist(s) - Album - Track number - Track title.flac

Example: Bill Evans Trio - Portrait In Jazz - 02 - Autumn Leaves.flac

I chose this scheme because it’s easily searchable and independent of the overlying folder structure.

Each album folder contains:

  • A log file (generated from either Exact Audio Copy or X Lossless Decoder).
  • A cue sheet file (for legacy purposes).
  • An image of the album cover.

The inclusion of a .log and .cue file applies only to albums ripped from a physical CD or hybrid SACD.

The vast majority of my collection is sourced from CDs. Vinyl rips are included only in cases where a CD release is unavailable or does not exist. Sometimes the album’s mastering is notably better on vinyl than its CD equivalent (e.g. the 2019 vinyl remaster of Flying Lotus’ Cosmogramma has significantly less dynamic range compression than previous versions). This database provides useful dynamic range data.

Why do I prefer CD rips over LP rips? When configured properly, Exact Audio Copy produces high quality rips that maintain the integrity of CD data.

My collection includes some web-sourced releases, most of which come from Bandcamp or Boomkat.

The included album artwork is generally high quality, at least 1000x1000 resolution minus some hard to find exceptions.

All files are in lossless format (FLAC) and tagged with the following vorbis comment metadata using Ex Falso:

  • Title
  • Track number
  • Artist
  • Album artist
  • Album
  • Label
  • Catalog number
  • Date
  • Release date
  • Disc number
  • Media (e.g. CD, SACD, LP, WEB)
  • ReplayGain settings (album gain and peak, track gain and peak)

Catalog and label information is particularly helpful when distinguishing between albums with multiple versions. E.g., Pink Floyd's The Dark Side Of The Moon has 1,100+ releases currently listed on Discogs. Is it the 1973 Yugoslavian vinyl release by Jugoton Records or the 1989 South Korean cassette by EMI?

I check the spectral integrity of all albums using spectral analyzing software. This can be done with audacity.

Everything is accessed locally using gmusicbrowser, a fantastic music library application created by Quentin Sculo.

All content is legally obtained, of course.
